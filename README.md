# 🛡️ Base Security Vault

Security is the foundation of every successful on-chain journey. This repository documents my personal framework for staying safe on **Base**.

## 🛡️ Security Checklist
- **Contract Verification:** Always checking source code before interaction.
- **Revoke Monitoring:** Regular audits of token approvals and permissions.
- **Scam Prevention:** Tracking and blacklisting suspicious phishing links/contracts.

## 🗝️ Principles
1. Never share private keys.
2. Use separate wallets for high-risk mints.
3. Verify twice, sign once.
